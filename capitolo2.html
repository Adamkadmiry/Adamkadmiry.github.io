Hardware

L'hardware sono tutte le parti tangibili di un dispositivo elettronico. Per esempio, in un computer, possono esserci il mouse, la tastiera, il monitor, la RAM, la ROM, il disco fisso, la scheda madre, il processore, la scheda video, e non sono nemmeno tutte le componenti hardware. Solitamente, si dividono in due principali categorie: input e output. L'input è ciò che inserisce dati nel sistema, mentre l'output elabora un risultato. Per esempio, quando usi il mouse per aprire un'applicazione, inserisci dei dati (input), e quando l'applicazione viene aperta, l'input è stato elaborato e restituisce un output. Tuttavia, non ci sono dispositivi che sono solo di input o solo di output; ci sono dispositivi che ricevono sia input che danno output. Per esempio, le memorie di massa, come le chiavette USB, e anche le stampanti.


Software

Il software è la parte non tangibile di un dispositivo che permette l'esecuzione di programmi e applicazioni attraverso il sistema operativo. In altre parole, mentre l'hardware è la parte fisica di un computer o di un altro dispositivo (come il processore, la memoria o lo schermo), il software è ciò che fornisce le istruzioni che dicono all'hardware cosa fare.
